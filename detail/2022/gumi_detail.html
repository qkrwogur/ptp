1.제품 이미지

구미시 대기 환경 측정기

2.주요기능

14개의 센서와 센서 데이터를 수집하여 서버와 데이터을 전송한다. ota 기능이 존재하고, 전송 주기를 변경 기능과 미 전송 저장 기능이 존재 한다.

3. 아키텍처 다이어그램

통신 아키텍처
ota 아키텍처

4.나의 역할

  • ota 기능 구현
    • → 서버와 통신한 데이터의 단말의 버전 정보를 전달 하여 현재 단말의 단말 버전과 서버에 전송
    버전 정보가 다르면 펌웨어 ota 시작
  • 신규 센서 추가
    • → uart 통신 센서 추가
  • 미 전송 데이터 읽고 쓰기
    • → flash 데이터 저장되는 전송 실패한 센서 데이터를 저장 하고 전송
  • 보드간 데이터 연동
