서울시 공공 와이파이 트랙커

1. 제품 이미지

2. 주요기능

서울시 공공 와이파이 트랙커 단말로 주변 와이파이를 스캔하여 데이터를 서버로 전송. 와이파이 위치로 실종자의 위치를 확인하는 시스템



3. 아키텍처 다이어그램

4. 나의 역할

  • 명령어 데이터 처리
    • → usb로 입력되는 AT command 처리 기능 구현. 사용자가 입력하는 설정 값(wifi 설정, 목적 IP)을 저장하여 인터페이스 함수 구성
  • usb 충전 인터럽트 기능
    • → usb pin에 들어오는 high 신호를 인터럽트 처리. 인터럽트 high 이후 adc 측정으로 단말의 배터리 잔량을 led에 표시
  • peap 서버 테스트 구성
    • → 테스트를 위해 서울시에서 사용하고 있는 peap wifi 서버를 라즈베리 환경에 구성하고, Freeradius를 사용한 peap 서버를 구성
  • nvm3 저장 기능 분석 및 구성
    • → Silicon Labs에서 개발한 비휘발성 메모리 관리 시스템으로, nvm3의 활영 방법을 연구하고 사용. repacking 기술을 이용한 데이터 관리
